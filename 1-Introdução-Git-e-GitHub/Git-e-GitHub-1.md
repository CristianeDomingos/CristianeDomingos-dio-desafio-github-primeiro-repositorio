# Comandos de navegação 

#### Windows

- Lista - DIR
- Escolher Diretório  - CD/
- Voltar - CD ..
- Limpar - CLS
- Criar Pasta - MKDIR (nome da pasta)
- Deletar Arquivo - > DEL
- Criar Arquivo - ECHO
- Deletar Pasta - RMDIR (nome da pasta) /s /g 

#### Linux

- Lista - LS
- Escolher Diretório  - CD/
- Voltar - CD ..
- Limpar - CLEAR ou CTRL+L
- Criar Pasta - MKDIR (nome da pasta)
- Deletar Arquivo - > DEL
- Criar Arquivo - ECHO
- Deletar Pasta - RM - RF (nome da pasta)
